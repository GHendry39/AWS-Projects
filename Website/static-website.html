Static Website

Description

This website is hosted as a static website in an Amazon S3 bucket and published via an Amazon CloudFront distribution. The domain name is managed by Amazon Route 53 with SSL certificate issued by AWS Certificate Manager.

Architecture Diagram

Services

Amazon Route 53

Amazon Route 53 is a Domain Name Server (DNS) that translates internet URLs to IP addresses for websites/web apps hosted by AWS. The website domain was registered in Route 53.

Amazon S3

Amazon S3 (Simple Storage Solution) is an object storage service that stores data in 'buckets'.

Amazon CloudFront

Amazon CloudFront is a Content Delivery Network (CDN) that distributes the website to Edge location all over the world. This ensures low latencies for all users.

AWS Certificate Manager

To make the website access secure, we need an SSL certificate. This is supplied by the AS Certificate manager
